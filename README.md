# OLKUN SİNEKLİK Teklif Sistemi

Vercel uyumlu statik web uygulaması.

## Kurulum

1. Bu klasörü GitHub'a yükleyin.
2. Vercel hesabınızda "Add New Project" seçin.
3. GitHub reposunu seçin.
4. Deploy butonuna basın.

## Özellikler

- Çoklu ölçü girişi
- Beyaz / Renkli
- Standart / Duble
- Bayi / Son kullanıcı modu
- WhatsApp teklif gönderme
- PDF / Yazdır
- Mobil uyumlu saha ekranı
